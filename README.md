# AI Market Database Agent

A quantitative finance and data engineering project that builds a market data pipeline using **Python, MySQL, and Streamlit**.

## Features

- Download market data from Yahoo Finance
- Store financial data in MySQL
- Compute quant indicators (MA50, MA200, Volatility)
- Portfolio analytics
- Monte Carlo portfolio optimization
- Sharpe ratio analysis
- Natural language SQL queries
- Interactive Streamlit dashboard

## Tech Stack

- Python
- Streamlit
- MySQL
- Pandas
- NumPy
- Matplotlib
- Yahoo Finance API
